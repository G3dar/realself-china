# Real Self China Microsite

Microsite for Real Self immersive experience - China partnership opportunities.

## Live URLs

- **Production**: https://realself.asia
- **WWW**: https://www.realself.asia
- **Vercel**: https://realself-china.vercel.app

## Deployment

Deploys automatically to Vercel on push to `master` branch.

### Manual Deploy

```bash
cd realself-china
vercel --prod
```

### Vercel Project Info

- **Team**: g-3darcoms-projects
- **Project**: realself-china
- **Root Directory**: realself-china/
- **GitHub Repo**: G3dar/realself-china

## Local Development

```bash
# Start local server
python -m http.server 8080

# Open http://localhost:8080/home.html
```
